SSH Terminal

Getting Started

1. On your server, clone and start the backend: git clone github.com/Ashosystem/r1-ssh-terminal cd r1-ssh-terminal && npx r1-terminal
2. Copy the URL and token printed in your terminal
3. Tap Continue and enter them to connect

Setup

Enter your backend URL and auth token. Run the backend server first.
